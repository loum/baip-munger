Two examples of bulleted lists follow. If the dot points are full sentences:

·       Each should start with a capital letter. [style = List Bullet]

·       Each should end in a full stop.

If the dot points are sentence fragments, follow these conventions:

·       no capitals at beginning

-      second level bullet [style = List Bullet 2]

-      second level bullet

·       no ‘and’ after the second to last dot point

·       no full stop at the end of each dot point (except for the last).
